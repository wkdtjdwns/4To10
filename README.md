# 4 = 10

모바일 게임 "4=10"을 클론코딩한 프로젝트. (WPF / .NET 10 사용)<br>
**4=10**: 4개의 숫자와 사칙연산·괄호로 **10을 만드는** 퍼즐 게임.

- 숫자 4개(중복 가능)와 기호 `( + − × ÷ )`
- 모든 연산을 다 쓸 필요는 없고, 같은 연산을 여러 번 써도 됨. 괄호는 **한 쌍만** 사용 가능.
- 수식의 결과가 **정확히 10**이 되면 정답. (나눗셈은 소수로, 최대 3자리까지 표시 — 예: `10 ÷ 3 = 3.333...`)
- 25개 스테이지 · 진행상황 자동 저장 · 힌트(숫자 순서 / 연산자 순서)

## 실행 방법

### 방법 1 — 그냥 실행 (권장, .NET 설치 불필요)

[Releases](../../releases)에서 **`FourTen.exe`** 를 **클릭해서 다운로드** 후에 실행하면 됨. (별도 설치 없이 바로 실행)


<br>

### 방법 2 — 소스에서 실행

1. [.NET 10 SDK](https://dotnet.microsoft.com/download) 설치
2. 프로젝트 폴더에서:
   ```
   dotnet run
   ```
   또는 Visual Studio 2022에서 `FourTen.slnx` 를 열고 **F5**.

## ⚠️ 실행이 안 되고 "애플리케이션 제어 정책에서 이 파일을 차단했습니다 (0x800711C7)" 가 뜬다면?

이건 앱 문제가 아니라 **Windows의 Smart App Control(스마트 앱 컨트롤)** 이 켜져 있기 때문임.<br>
이 기능은 서명되지 않은 앱을 차단하는데, 개인이 만든 앱은 여기에 해당함...

**해결 방법**
1. `Win + R` → `windowsdefender://smartappcontrol` 입력 → Enter
2. **스마트 앱 컨트롤 → 끄기**
3. **컴퓨터 재부팅** (끄기는 재부팅해야 완전히 적용됨)

> 참고: Smart App Control은 한 번 끄고 다시 켜려면 Windows를 재설정해야 함.

## 조작법

| 대상 | 방법 |
|------|------|
| **연산자** `+ − × ÷` | 팔레트에서 클릭 후 숫자 사이 칸 클릭, 또는 **드래그**해서 칸에 놓기 |
| 연산자 이동/삭제 | 놓인 연산자를 다른 칸으로 **드래그**(이동) / 밖으로 **드래그**하거나 클릭(삭제) |
| **괄호** `( )` | 클릭 후 숫자 클릭, 또는 **드래그**해서 숫자에 놓기 (이동·삭제도 클릭 or 드래그) |
| **숫자 순서** | 두 숫자를 차례로 탭(교환), 또는 숫자를 다른 숫자로 **드래그** |
| 힌트 | 좌측 하단 `?` |
| 초기화 | 우측 하단 `↻` |
